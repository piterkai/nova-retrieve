# 产品 FAQ —— SmartDoc 智能文档平台

## 计费与套餐

**Q: 免费版与企业版有什么区别？**
免费版限制 5 名用户、1GB 存储、不支持 SSO 与审计日志。企业版无人数上限、按存储分级计费，支持私有化部署与 SLA 99.95%。

**Q: 可以按月付费吗？**
个人专业版支持月付（39 元/月）和年付（388 元/年，约 8.3 折）。企业版仅支持年付，最少 10 个席位起售。

**Q: 学生有优惠吗？**
有。在校学生凭 .edu 邮箱可申请个人专业版 50% 折扣，每年需复验一次。

## 数据与安全

**Q: 数据存储在哪里？**
默认存储在阿里云华东 2（上海）region。企业客户可选华北 2（北京）、新加坡 region；私有化部署完全部署在客户机房。

**Q: 是否符合等保 2.0？**
SmartDoc 已通过等保 2.0 三级认证，并通过 ISO 27001 和 SOC 2 Type II 审计。

**Q: 误删的文档能恢复吗？**
所有删除操作进入回收站保留 30 天；企业版可启用版本快照，最长保留 180 天历史版本。

## 集成与 API

**Q: 是否提供 OpenAPI？**
是。REST API 文档位于 https://docs.smartdoc.example/api，速率限制为 600 次/分钟。
Webhook 支持文档创建/修改/删除三类事件。

**Q: 支持哪些 SSO？**
企业版支持 SAML 2.0、OIDC、企业微信、飞书、钉钉一键登录。
